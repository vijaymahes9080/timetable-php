# Automatic Timetable Generator

A modern Web Application designed to automatically generate timetables for educational institutions.

## Features

- **Automated Generation**: Generate schedules efficiently without overlaps or conflicts.
- **Admin Dashboard**: Comprehensive management of teachers, courses, students, and semesters.
- **Staff Dashboard**: Allows faculty members to view their individual schedules.
- **Student Dashboard**: Provides students with easy access to their semester timetables.
- **Responsive Interface**: Sleek UI designed for both desktop and mobile platforms.

## Installation & Setup

1. **Prerequisites**: Ensure you have PHP, Apache, and MySQL installed (e.g., via XAMPP, WAMP, or Laragon).
2. **Database Import**: Import the SQL schema from the `database/` directory into your MySQL instance.
3. **Configuration**: Edit `config.php` to update your database credentials.
4. **Run Application**: Place the project folder in your web server's root directory (e.g., `htdocs` or `www`) and access it via your web browser.
